# DocNerve Backend Requirements
# Install: pip install -r requirements.txt
#
# ── IMPORTANT: llama-cpp-python GPU setup (RTX 40 series / Ada Lovelace) ────
#
#   The standard PyPI build of llama-cpp-python does NOT include CUDA support
#   on Windows.  After running `pip install -r requirements.txt`, replace the
#   CPU wheel with the pre-built CUDA Ada (sm_89) wheel:
#
#   pip install "https://github.com/dougeeai/llama-cpp-python-wheels/releases/download/v0.3.16-cuda12.1-sm89-py312/llama_cpp_python-0.3.16+cuda12.1.sm89.ada-cp312-cp312-win_amd64.whl" --force-reinstall
#
#   Requirements for the CUDA wheel:
#     - GPU   : NVIDIA RTX 4060 / 4060 Ti / 4070 / 4080 / 4090  (Ada Lovelace)
#     - Driver: 591.xx or newer  (nvidia-smi shows "12.x" CUDA capability)
#     - CUDA Toolkit 12.1 must be installed on the system:
#         https://developer.nvidia.com/cuda-12-1-0-download-archive
#     - Python 3.12.x  (exact minor version must match the wheel tag cp312)
#
#   Without the CUDA toolkit the GGUF models will run on CPU only (slow but
#   fully functional — the pipeline falls back gracefully).
# ─────────────────────────────────────────────────────────────────────────────
